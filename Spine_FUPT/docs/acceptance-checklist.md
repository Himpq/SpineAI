# 脊柱AI影像随访系统 — 功能验收清单

> 基于路线图第 1 阶段测试步骤整理 · 2026-03-20  
> 使用方式：在真机上逐条操作，通过则打 ✅，未通过填写问题描述

## 流程1：医生日常工作流

### M1.1 登录/登出

- [ ] 输入错误密码 → 显示"账号或密码错误"
- [ ] 输入正确账号密码 → 成功进入首页
- [ ] 点击退出 → 回到登录页
- [ ] 切后台再回来 → 仍保持登录态（不会闪退或需重登）

### M1.2 总览仪表板

- [ ] 登录后自动加载仪表板数据（患者总数、待复核数、未读消息等）
- [ ] 下拉刷新 → 数据更新
- [ ] 仪表板数字与实际数据对应（可交叉验证后台数据库）

### M1.3 患者列表+搜索

- [ ] 患者列表正常加载，显示姓名、状态标签
- [ ] 输入关键字搜索 → 列表过滤出匹配患者
- [ ] 点击患者卡片 → 进入患者详情页

### M1.4 患者详情

- [ ] 基本信息正确显示（姓名、年龄、性别、手机号）
- [ ] 时间线有数据且按时间倒序排列
- [ ] Cobb 角趋势图表正常渲染（有检查记录时）
- [ ] 检查列表可点击查看详情

### M1.5 复核队列+详情

- [ ] 复核队列正常加载
- [ ] 状态筛选（待复核/已复核/全部）正确过滤
- [ ] 点击进入复核详情页
- [ ] X 光原图与 AI 推理结果图可查看
- [ ] 提交复核报告（填写备注 → 点击确认） → 状态变更为"已复核"

### M1.6 病例分享

- [ ] 生成外部分享链接 → 链接可复制
- [ ] 分享链接页面可打开（显示病例详情）
- [ ] 站内分享给其他用户 → 对方在聊天中收到分享消息
- [ ] 分享链接页面可添加评论

### M1.7 用户管理

- [ ] 管理员可查看用户列表
- [ ] 创建新用户（填写用户名/密码/角色）→ 创建成功
- [ ] 编辑用户角色/状态 → 保存生效
- [ ] 非管理员账号无用户管理入口

### M1.8 设置/主题

- [ ] 切换深色/浅色主题 → 界面切换
- [ ] 修改服务器地址 → 保存后生效
- [ ] 清除缓存 → 操作成功

## 流程2：患者自助随访

### M2.1 门户进入

- [ ] 使用患者 portal_token 进入门户页面
- [ ] 显示个人信息（姓名、年龄、性别）
- [ ] 显示最新检查记录

### M2.2 上传影像

- [ ] 拍照/选图上传影像 → 上传成功提示
- [ ] 等待 AI 推理完成
- [ ] 查看 AI 推理结果（分类、Cobb 角等）

### M2.3 随访时间线

- [ ] 查看所有历史检查记录列表
- [ ] 点击单条记录查看详情

### M2.4 门户登出

- [ ] 点击退出 → 回到登录/扫码页面
- [ ] 再次进入需重新扫码/输入 token

## 流程3：问卷随访

### M3.1 创建/编辑问卷

- [ ] 创建新问卷（包含文本题+选择题）→ 创建成功
- [ ] 编辑已有问卷 → 修改保存成功
- [ ] 问卷预览显示正确

### M3.2 分配问卷

- [ ] 选择患者分配问卷 → 分配成功
- [ ] 患者端聊天中收到问卷链接消息

### M3.3 患者填写

- [ ] 通过链接打开问卷页面 → 题目正常显示
- [ ] 填写所有题目并提交 → 提交成功
- [ ] 重复提交 → 被拦截（显示"已提交"）

### M3.4 查看结果

- [ ] 查看已回收问卷列表 → 显示填写者和时间
- [ ] 查看单份回答详情 → 各题答案正确展示
- [ ] 终止问卷收集 → 状态变更为"已终止"
